# Changelog

All notable public versions of `Arkanoid: Revenge of the Dope` are tracked here.

## v1.0.4

- Added social sharing metadata for GitHub Pages so LinkedIn and other platforms can show a proper preview image
- Added a dedicated social preview image asset for shared links
- Updated the GitHub Pages workflow to opt into the newer JavaScript action runtime
- Tightened GitHub-side project metadata and repository polish

## v1.0.3

- Fixed trapped-ball jitter around tight brick and indestructible block layouts
- Adjusted the stage two side opening to be less punishing

## v1.0.2

- Replaced the title music with the new track
- Moved the previous title music into gameplay rotation as track 5

## v1.0.1

- Packaged the current public build with README and license
- Refreshed release packaging and public project milestone tracking

## v1.0.0

- First public playable release
- Browser-playable Arkanoid tribute with title video, soundtrack rotation, enemies, power-ups, boss fight, and GitHub Pages deployment
